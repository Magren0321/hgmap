# 华广地图
华广地图是基于Vue + Typescript + Element ui 技术栈由学生独立开发的web客户端。旨在帮助师生了解学校设施位置以及帮助新生快速熟悉校园。

同时该项目也是个人在学习vue结合typescrpt中所写的小demo，不足的部分欢迎指出并[反馈](https://github.com/Magren0321/hgmap/issues) :v:
当然也欢迎Star！ :star:
